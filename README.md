Добро пожаловать в мой мини-проект! Этот бот создан для того, чтобы быстро и удобно представлять своё портфолио: навыки, 
проекты и контакты — всё в одном месте. 
Бот позволяет показывать информацию обо мне, отображать мои навыки
, выводить список проектов и отправлять ссылки на соцсети. Я постарался сделать его простым, аккуратным и приятным в использовании.
